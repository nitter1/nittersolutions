Principais Dúvidas (FAQ)

O que é necessário para obter um site?

Domínio: Endereço eletrônico do site (ex: www.empresa.com.br), adquirido no registro.br ou outra empresa de registro de domínios.

Hospedagem: Local onde os arquivos do site e os emails profissionais são armazenados.

Logotipo: Identidade visual fundamental para o design do site, oferecemos serviços de criação de logotipos profissionais.

O site possui alguma mensalidade?

Nós cobramos apenas pela criação do site. Manutenções mensais ou avulsas podem ser contratadas separadamente, sem obrigatoriedade.

Quanto custa um site?

O custo de um site varia de acordo com as necessidades e objetivos de cada cliente. Cada projeto é único, com diferentes níveis de complexidade e tamanho. Realizamos um orçamento personalizado, considerando esses aspectos específicos para garantir um resultado que atenda perfeitamente às expectativas e requisitos de cada cliente.

Qual é o prazo de entrega?

Tempo médio: Em até 15 dias úteis para o site básico, e em até a 45 dias úteis para o site avançado.

O site é editável? Pertence ao cliente?

Sim. Após a conclusão do site, enviaremos os dados de acesso ao painel administrativo para que você possa editar o conteúdo de forma fácil e intuitiva. Se desejar, também podemos enviar os arquivos de backup.

O site aparecerá no Google?

Sim, nossos sites são otimizados para o Google, entretanto, para alcançar o topo das pesquisas, especialmente em mercados competitivos, recomendamos a contratação de uma agência especializada em marketing ou publicidade. Elas oferecem serviços mais abrangentes e elaborados, garantindo resultados ainda mais eficazes.

Qual a diferença entre os planos Básico e Avançado?

1. Básico: Site estilo "one page" (página única), pronto em até 7 dias úteis, com até 5 seções resumidas (ex.: Banner, Quem Somos, Serviços, Galeria, Contato). Ideal para Google ADS, produtos específicos e novas empresas.

2. Avançado: Site institucional exclusivo, sob demanda, com recursos ilimitados. Ideal para empresas pequenas e médias buscando publicidade, melhor posição no Google, credibilidade e engajamento.

Projeto

1. Primeiramente, realizamos uma conversa detalhada para entender completamente o projeto e enviar o orçamento ao cliente, visando uma comunicação clara e eficaz.

2. Após a aprovação do orçamento, o cliente é solicitado a efetuar o depósito do sinal de 50% para que possamos iniciar o projeto imediatamente, garantindo uma colaboração fluida e produtiva.

Planejamento

3. Solicitamos ao cliente o envio dos materiais básicos, como logotipo (se disponível), textos (redação) e imagens reais de seu negócio e a seleção dos principais tópicos/seções para o site. Além disso, é necessário fornecer os dados para o registro do domínio.

4. No plano avançado, criamos um layout inicial que é enviado ao cliente para aprovação antes da construção no ambiente web.

Nos planos básico e avançado, oferecemos modelos (templates) para que o cliente escolha.

Entrega

5. Iniciamos o desenvolvimento do site e realizamos os testes necessários.

6. Entregamos o projeto concluído.

7. Fornecemos um manual simples e objetivo para ajudar o cliente na administração do site.
